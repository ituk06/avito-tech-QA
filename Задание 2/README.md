# Задание 2.1: Тестирование API микросервиса объявлений

## Описание

**Host**: `https://qa-internship.avito.com`
API версии: `/api/1/...` и `/api/2/...`

- Python ≥ **3.8**

- `pip` 

## Установка и настройка
1. Склонируйте репозиторий
- git clone [https://github.com/ituk06/avito-tech-QA](https://github.com/ituk06/avito-tech-QA.git)](https://github.com/ituk06/avito-tech-QA.git)

2. Перейдите в директорию проекта
- cd avito-tech-QA

3. Установка зависимостей
- python -m venv venv
- pip install -r requirements.txt

5. Запустите тесты
- pytest test_api.py -v
